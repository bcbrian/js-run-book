## How to Keep it DRY(Dont Repeat Yourself)

### Functions are a way

```js
// basic
function mySpecialFunction() {
  // do something cool!
}
mySpecialFunction(); // execute order 626
// returns will stop execution
function returns(){
  // some code here
  alert("You will see me");
  return;
  alert("You wont see me");
  // and code here won't run
}

// returns will send info back if provided
function sum(oneParam, twoParam){
  return oneParam + twoParam
}
// this way 
const six = sum(3, 3)
alert(six); // 6
// this way ... samesies
let oneParam = 3;
let twoParam = 3;
const answer = sum(oneParam, twoParam)
alert(answer); // 6

let oneParam = 21;
let twoParam = 21;
const answerTwo = sum(oneParam, twoParam)
alert(answer); // 42 it is always the right answer...
```
### Arrow Functions
They 90% the same as above just use less letters to write and have an option to do an implicit return.
> 10% difference coming later 

```js
// implicit return
const sum = (oneParam, twoParam) => oneParam + twoParam 
// not implicit return
const sum = (oneParam, twoParam) => {
  return oneParam + twoParam
} 
```

## Loops
### Forloop
Loop for a specified duration
```js

  const element = array[index];
  // do stuff with element
}
```

### Array Methods
#### Array.map

This is a method that loops and operates on one array element at a time. When it is all done it returs a new array based on what each iteration returned.
```js
/**
 * const people = [{
 *   age: 23,
 *   heigh: 6,
 *   weight: 197
 * }, ...]
 * */

const ages = people.map((person) => person.age);

/**
 * ages = [23, ...]
 */
```
